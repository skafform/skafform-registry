`@skafform/lite-docs` est un brick de documentation pour les projets skafform. Il transforme des fichiers MDX en un site de documentation avec sidebar auto-générée, table des matières et recherche full-text.

## Fonctionnalités

- Fichiers MDX avec frontmatter
- Sidebar générée automatiquement depuis la structure des dossiers
- Table des matières par page
- Recherche full-text côté client
- Hot reload en développement
- Zéro configuration

## Démarrage rapide

```bash
skafform add @skafform/lite-docs
```

Crée un dossier `docs/` à la racine du projet et ajoute des fichiers `.mdx`.
